# Playwright Component Testing Example

This is an example of using [Playwright](https://playwright.dev/) for component testing. It uses an editable text component from Ark UI as the component under test.

## Steps for blog post

1. Init a playwright component testing setup... pnpm create playwright --ct 
2. Add Ark UI... pnpm add @ark-ui/react
3. Create a component... src/components/EditableText.tsx
4. Create a test file... src/components/EditableText.test.tsx
5. Adjust package.json scripts...
